# 🎙️ AI Audio to Text Converter

A simple Audio to Text Converter built using **Spring Boot**, **Spring AI**, **OpenAI**, and **React**. Users can upload an audio file and get the transcribed text instantly.

## 🚀 Features

- Upload audio files
- Convert speech to text using OpenAI
- Simple React frontend
- Spring Boot REST API
- Fast and easy to use

## 🛠️ Tech Stack

### Backend
- Java
- Spring Boot
- Spring AI
- Maven

### Frontend
- React
- Axios
- CSS
